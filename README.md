# Labor Forecast Dashboard

A Python project that simulates hourly demand, forecasts future demand, and converts forecasts into staffing recommendations.

## Current Progress
- Synthetic hourly demand generator
- Time-series components: seasonality, trend, and noise

## Tech Stack
- Python
- Pandas
- NumPy
- Streamlit
- Matplotlib

## Run

Generate the synthetic dataset:

```bash
python src/data_generator.py
